# TickTick Integration Reference

## Intent
Use TickTick MCP tools to:
- list projects/tasks,
- create/update/complete tasks,
- fetch current workload for summaries.

## Mapping guidance
- `p1/p2/p3/p4` in notes should map to high/medium/normal/low priority.
- Keep task titles short and actionable.
- Add due dates only when date intent is explicit.

## Safety
- Check for likely duplicates before creating many tasks.
- Return task identifiers or links in phase output when available.
